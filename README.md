# Scalap

* Trees
    * KDTree
* Hashing
    * Bloom Filters
    * MinHash
    * LSH

## Usage

    Usage: ./exe <options>
    Options:
       --test         run all tests
       --examples     run all examples
       --kdtree       run KDTree example
       --bloomfilter  run Bloom Filter example
       --minhash      run MinHash example
       --lsh          run LSH example
    
## Compile 
     
    sbt package
    
 creates a jar file in target directory
